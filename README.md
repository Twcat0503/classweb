## DLEC 9th 第九屆數位學習實驗班
[DLEC 9th web](https://dlec9th.twcat0503.org)

用於介紹數位班、學生成果發表會、成員與相關資訊。

### 技術棧
- Next.js 13 (App Router)
- Tailwind CSS
- TypeScript
- ESLint

### 開發啟動
```bash
npm run dev
```

### 品質檢查與建置
```bash
npm run lint  
npm run build  
npm start      
```

### 結構
- app/: 使用 App Router，各頁面已預留段落與占位圖示
- components/: 站點框架、主題切換、動畫載入
- public/: 可放置logo與專題圖片

### 部署
部署至 Cloudflare Pages 子網域 dlec9th.twcat0503.org
